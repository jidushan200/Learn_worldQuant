##  Alpha Examples for Beginners

### 1.营业收入年度表现operating_income

```vala
# ========== Simulation Settings ==========
# Region        : USA
# Universe      : TOP3000
# Language      : Fast Expression
# Decay         : 0
# Delay         : 1
# Truncation    : 0.08
# Neutralization: Subindustry
# Pasteurization: On
# NaN Handling  : Off
# Unit Handling : Verify
# Max Trade     : OFF
# Max Position  : OFF
# =========================================

# 原版：直接比较营业收入的历史表现
# ts_rank(operating_income, 252)

# 改进1：加入股价，变成营业收益率
# ts_rank(operating_income / close, 252)

# 改进2：用市值代替股价，更合理地反映公司规模
# 因为股价不能代表公司大小，市值（cap = 股价 × 流通股数）才能真正反映公司规模

【已submit过ts_rank(operating_income / close, 252)，需要做查重操作】ts_rank(operating_income / cap, 252)
```

```vala
# 原版：252天窗口时序排名
# -ts_rank(fn_liab_fair_val_l1_a, 252)

# 尝试用delta替代rank，Sharpe崩溃，放弃
# -ts_delta(fn_liab_fair_val_l1_a, 63)

# 缩短窗口至63天，Sharpe 1.14，未通过
# -ts_rank(fn_liab_fair_val_l1_a, 63)

# 加入截面rank外层，Sharpe仍为1.14，未通过
# -rank(ts_rank(fn_liab_fair_val_l1_a, 63))

# 窗口调整至84天，Sharpe 1.38，通过
# rank()外层截面排名是核心突破点，84天是最优甜蜜点

【已提交】-rank(ts_rank(fn_liab_fair_val_l1_a, 84))
```
