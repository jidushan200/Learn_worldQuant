### 背调前：常见入行疑问 & 获取背调协议表

**Q1：为什么我提交了因子，还没有更新积分（挑战分）？**

A：平台提交因子刷新时间按照美东时间 0 点计算：

- 夏令时：中国时间 **12:00** = 美东次日 0 点
- 冬令时：中国时间 **13:00** = 美东次日 0 点
- 每日挑战分于中国时间 **14:00–15:00** 刷新
- 即第 N 天提交 → 第 **N+1** 天 14–15 点才刷新

> 补充：每天提交 1 个因子约 1900+ 分，2 个即封顶 2000 分，无需为交而交，**以收到背调协议表为主**。
>
> 积分查看：顶部导航栏 → `Competition（比赛）` → `Challenge（挑战）`

------

**Q2：已达一万分且到达1万分后下午没收到邀请，仍未收到背调申请表？**

A：检查是否有 Workday 入口（小人图标 → Account），没有则联系官方邮箱：`mainlandchina@worldquantbrain.com`

**Q4：账户被封禁了怎么办？**

A：先排查原因：

1. 泄露账户密码（如 GitHub）→ **无法解封**
2. 只回测不提交 / 低回测高提交 → 可通过**面试解封**
3. 团队风控（买卖因子），查实后整个团队均受影响 → 可通过**面试解封**
4. 顾问阶段 Regular 因子平均相关性 > 0.7 且持续过久 → 可通过**面试解封**

> 第 2、3、4 条：面试内容将发送总部审核。

------

**Q5：顾问合同审核太慢，有没有加速方法？**

A：

- 提交 **≥ 20 个因子** 进入高优先级审核队列，按时间顺序处理
- 在**中文论坛发帖**可进一步提升审核进度

### 背调篇

**Q1：背调可以不打电话联系我 / 不联系公司吗？**

A：在背调表中找到以下字段填写 **YES**：

> *Please indicate if we should be cautious about reaching you at any of the above numbers (e.g., current work number): (Required)*

- 填 YES → 在职人员仅**邮件**联系，不打电话
- 学生身份 → **不会被联系**

------

**Q2：背景调查具体查什么内容？**

A：按照填写的资料表内容进行核查，目前重点审查**犯罪记录**，非金融机构相关工作一般均可通过。

在职人员若选择不打扰公司，需自行提供以下材料之一（**三选一**）：

- 在职证明
- 劳动合同
- 社保缴纳记录

> 法人代表等相关身份也会被审查，非金融公司背景同样可通过。
>  审核结果（通过 / 不通过）均会**邮件通知**。

------

**Q3：IQC 排名太低，如何提升？**

A：薪酬与排名主要由以下维度决定：

- **Sharpe、Fitness、Margin** → 越高越好
- **自相关性** → 越低越好
- **多类别覆盖** → 跨地区、跨品类提交
- **D0 + D1** → 合理搭配不同持仓周期因子
