##  Alpha Examples for Beginners

### 1.营业收入年度表现operating_income

```vala
# ========== Simulation Settings ==========
# Region        : USA
# Universe      : TOP3000
# Language      : Fast Expression
# Decay         : 0
# Delay         : 1
# Truncation    : 0.08
# Neutralization: Subindustry
# Pasteurization: On
# NaN Handling  : Off
# Unit Handling : Verify
# Max Trade     : OFF
# Max Position  : OFF
# =========================================

# 原版：直接比较营业收入的历史表现
# ts_rank(operating_income, 252)

# 改进1：加入股价，变成营业收益率
# ts_rank(operating_income / close, 252)

# 改进2：用市值代替股价，更合理地反映公司规模
# 因为股价不能代表公司大小，市值（cap = 股价 × 流通股数）才能真正反映公司规模

【已submit过】ts_rank(operating_income / cap, 252)
```
