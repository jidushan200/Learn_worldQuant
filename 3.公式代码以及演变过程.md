### 1.营业收入年度表现 operating_income

```vala
# ========== Simulation Settings ==========
# Region        : USA
# Universe      : TOP3000
# Language      : Fast Expression
# Decay         : 0
# Delay         : 1
# Truncation    : 0.08
# Neutralization: Subindustry
# Pasteurization: On
# NaN Handling  : Off
# Unit Handling : Verify
# Max Trade     : OFF
# Max Position  : OFF
# =========================================

# 原版：直接比较营业收入的历史表现
# ts_rank(operating_income, 252)

# 改进1：加入股价，变成营业收益率
# ts_rank(operating_income / close, 252)

# 改进2：用市值代替股价，更合理地反映公司规模
# 因为股价不能代表公司大小，市值（cap = 股价 × 流通股数）才能真正反映公司规模

【已submit过ts_rank(operating_income / close, 252)，需要做查重操作】ts_rank(operating_income / cap, 252)
```

### 3.杠杆的力量 Power of Leverage

```vala
# ========== Simulation Settings ==========
# Region        : USA
# Universe      : TOP3000
# Language      : Fast Expression
# Decay         : 0
# Delay         : 1
# Truncation    : 0.01
# Neutralization: Subindustry  ← Market 改为 Subindustry（关键！）
# Pasteurization: On
# NaN Handling  : Off
# Unit Handling : Verify
# Max Trade     : OFF
# Max Position  : OFF
# =========================================

# 演变过程：
# Step 1: 原版 liabilities/assets + Market中性化 → Sharpe 0.77 ❌
#         不同行业负债率差异巨大，Market中性化无法消除行业噪音
#
# Step 2: 改为 Subindustry中性化 → Sharpe 1.54 ✅
#         同子行业内比较，信号更纯净
#
# Step 3: 尝试 liabilities/sqrt(cap * assets) → Sharpe 1.33 ✅
#         分母引入市值，账面+市场双视角衡量杠杆
#         有单位WARNING但不影响提交

# 最终答案：混合杠杆率
# 负债 ÷ (市值×总资产的几何平均)
# 同子行业内，相对负债越重的公司，短期越跑赢
【已提交：2026-04-14 17:56】liabilities/sqrt(cap * assets)
```
